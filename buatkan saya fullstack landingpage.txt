 ## Ringkasan Aplikasi Landing Page Agency Property

  Teknologi yang Digunakan
   - Backend: Laravel 12
   - Frontend: Tailwind CSS
   - Admin Panel: Filament
   - Development Environment: Laragon
   - Database: MySQL

  Alur Aplikasi

   1. Homepage/Landing Page
      - Hero section dengan CTA untuk menjelajahi properti
      - Preview properti unggulan
      - Navigasi ke berbagai kategori properti (Perumahan, Ruko, Apartemen, Vila)
      - Informasi tentang agency
      - Form kontak

   2. Navigasi Properti
      - Filter berdasarkan kategori (Perumahan, Ruko, Apartemen, Vila)
      - Filter berdasarkan lokasi, harga, dan spesifikasi
      - Tampilan grid/card properti dengan gambar dan informasi dasar

   3. Detail Properti
      - Galeri foto properti
      - Spesifikasi lengkap (jumlah kamar, luas tanah/bangunan, fasilitas)
      - Deskripsi detail properti
      - Lokasi pada peta
      - Tombol CTA (Hubungi Agent, Download E-brochure)

   4. Download E-brochure
      - Pengunjung dapat mengunduh brosur digital properti
      - Form singkat untuk mendapatkan brosur (nama, email, no telepon)

   5. Contact Agency
      - Form kontak lengkap (nama, email, telepon, pesan)
      - Informasi kontak agency (alamat, telepon, email)
      - Peta lokasi kantor agency

   6. Admin Panel (Filament)
      - Manajemen properti (tambah, edit, hapus)
      - Manajemen kategori properti
      - Manajemen konten landing page (hero section, about, services)
      - Melihat dan mengelola permintaan brosur
      - Melihat dan mengelola pesan kontak

  Fitur Utama

  Untuk Pengunjung:
   1. Explorasi Properti
      - Tampilan grid/card yang menarik untuk menjelajahi properti
      - Filter dan pencarian properti berdasarkan kriteria tertentu
      - Detail properti yang lengkap dengan gambar dan spesifikasi

   2. Download E-brochure
      - Tombol download brosur pada setiap halaman detail properti
      - Form singkat untuk mengakses brosur
      - Pengiriman brosur via email otomatis

   3. Contact Agency
      - Form kontak yang mudah diakses
      - Informasi kontak lengkap agency
      - Respon cepat dari tim agency

   4. Responsive Design
      - Tampilan yang optimal di desktop, tablet, dan mobile
      - Navigasi yang intuitif

  Untuk Admin Agency:
   1. Manajemen Properti
      - CRUD (Create, Read, Update, Delete) properti
      - Upload gambar dan dokumen properti
      - Kategorisasi properti (Perumahan, Ruko, Apartemen, Vila)

   2. Manajemen Konten
      - Edit konten landing page tanpa coding
      - Update informasi agency
      - Manajemen testimonial klien

   3. Manajemen Leads
      - Melihat daftar permintaan brosur
      - Melihat dan merespon pesan kontak
      - Ekspor data leads

  Struktur Database Awal

   1. Properti
      - id, nama, deskripsi, kategori, harga, lokasi, spesifikasi, status, gambar

   2. Kategori Properti
      - id, nama, deskripsi

   3. Leads Brousur
      - id, nama, email, telepon, properti_id, tanggal_request

   4. Pesan Kontak
      - id, nama, email, telepon, subjek, pesan, tanggal_kirim
