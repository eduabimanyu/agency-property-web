## Project Title: Website Agency Popery dengan Laravel 12

## Ringkasan Proyek:
Buatkan website landing page untuk agensi "Popery" yang menampilkan portfolio, layanan, dan informasi kontak. Website harus memiliki admin panel untuk mengelola konten dan dibangun dengan stack teknologi berikut:

Laravel 12 sebagai framework backend
Filament PHP sebagai admin panel
Tailwind CSS untuk styling frontend
MySQL sebagai database

## Fitur yang Diperlukan:

## Frontend (Tailwind CSS):
Header dengan navigasi responsive
Hero section dengan headline dan CTA
Section layanan yang ditawarkan
Section portfolio/proyek sebelumnya
Section testimonial klien
Section tentang tim
Form kontak
Footer dengan informasi kontak dan social media

## Admin Panel (Filament):
Kelola halaman (Pages management)
Kelola portfolio items (CRUD dengan gambar)
Kelola property items (CRUD dengan gambar)
Kelola testimonial
Kelola layanan
Pengaturan umum website (site title, logo, dll)

## Fitur Teknis:
Responsive design
Form kontak dengan validasi dan email notification
Optimasi SEO dasar (meta tags, structured data)
Loading cepat dan optimasi gambar
